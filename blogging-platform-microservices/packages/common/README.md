# Blogging Platform Microservices - Common Package

This directory contains shared resources and utilities that can be used across different microservices in the blogging platform. 

## Structure

- **src/models/index.js**: This file exports common models or utilities shared across services.

## Usage

To use the common package in your microservices, ensure to import the necessary models or utilities from the `common` package.

## Installation

To install the dependencies for the common package, run:

```
npm install
```

## Contribution

Feel free to contribute by adding more shared utilities or models that can benefit the overall architecture of the blogging platform.
