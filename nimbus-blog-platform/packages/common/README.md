# Common Package Documentation

This directory contains shared utilities and types that are used across different services in the Nimbus Blog Platform. The common package is designed to promote code reuse and maintainability by centralizing common functionalities.

## Installation

To install the common package, run:

```
npm install
```

## Usage

You can import the common utilities in your services as follows:

```javascript
const { someUtilityFunction } = require('@nimbus-blog-platform/common');
```

## Contributing

If you wish to contribute to the common package, please follow the guidelines outlined in the main project README. Ensure that any shared code is well-documented and tested.
